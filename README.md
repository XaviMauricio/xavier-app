# XAVIER APP 🚀⚽

Sistema de gestión personal de Xavier Mauricio.

## Secciones
- 💰 **Dinero** — Control de deudas y gastos mensuales
- 🚀 **Negocios** — Seguimiento de emprendimientos 2026
- ✅ **Hábitos** — Itinerario diario y tareas
- ⏱️ **Fitness** — Cronómetro HIIT con sonidos

## Uso
Abre `index.html` directamente en el navegador, o accede via GitHub Pages:

👉 **https://xavimauricio.github.io/xavier-app**

## Datos
Toda la información se guarda en `localStorage` del navegador — persiste aunque cierres o reinicies el dispositivo.
